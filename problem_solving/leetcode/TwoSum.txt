Inputs 

array of integers 
target integer

Asumptions 

each target have exactly one solution
should not use each element twice



Pseudo code


variables

int[] numbers;
int target;
int[] result;
int i = 0;
int flag = False;

While ( i < numbers.length() and flag == False) {

    int j=i+1;
    
    while ( j < numbers.length() and flag == False) {
    
       if ( numbers[i] + numbers[j] == target) {
       
           result[0]=i;
           result[1]=j;
           flag = True;
           
       } 
       j=j+1;
    }
    i=i+1;

}
